# Блог — Домашнее задание №4

Курсовая работа по дисциплине «Серверная веб-разработка», блок 3 — собственный
backend-фреймворк на PHP с маршрутизацией.

## Что нового в ДЗ4

**Сделан динамический заголовок страницы (`<title>`).** Каждый контроллер сам
решает, какое имя страницы будет показано во вкладке браузера, и передаёт его
в шаблон через переменную `$title`. В `templates/header.php` подставляется
`$title ?? 'Мой блог'` — это значение по умолчанию для главной.

Изменения относительно ДЗ3:

- `templates/header.php` — `<title>` подставляется из переменной `$title`
  (с фолбэком «Мой блог»).
- `src/MyProject/Controllers/MainController.php` — `sayHello` и `sayBye`
  передают `title` (например, «Страница приветствия», «Страница прощания»).
- `src/MyProject/Controllers/ArticlesController.php` — `show` передаёт
  заголовок статьи как `title`, поэтому во вкладке отображается название
  открытой статьи.

## Структура проекта

## Запуск

**Apache (XAMPP / OpenServer / хостинг):**
1. Импортируйте `database.sql` в phpMyAdmin (создаст БД `blog`).
2. При необходимости поправьте логин/пароль БД в `src/settings.php`.
3. Положите содержимое проекта в корневую папку сайта.

**Встроенный PHP-сервер:**
```bash
php -S localhost:8000 router.php
```

## Проверка работы (обратите внимание на вкладку браузера)

- `/` — заголовок вкладки: «Мой блог» (значение по умолчанию).
- `/hello/Иван` — заголовок: «Страница приветствия».
- `/bye/Иван` — заголовок: «Страница прощания».
- `/articles/1` — заголовок: название статьи.
