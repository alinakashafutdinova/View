# ДЗ №4 — Настраиваемый `title` страницы

Блог на самописном MVC-фреймворке (по лекции к теме 3). Включает в себя всё из ДЗ3 (роуты `/hello` и `/bye`) плюс задание ДЗ4.

## Что сделано по заданию

Заголовок страницы (`<title>`) теперь задаётся через переменную шаблона. Если `title` не передан — выводится значение по умолчанию **«Мой блог»**. Для страницы `/hello/username` задан title **«Страница приветствия»**.

Как это работает:
- В шаблоне `templates/header.php` тег title выводится так:
  ```php
  <title><?= $title ?? 'Мой блог' ?></title>
  ```
  Оператор `??` подставляет «Мой блог», если переменная `$title` не передана.
- Экшн `MainController::sayHello()` теперь рендерит шаблон `templates/main/hello.php` и передаёт в него `'title' => 'Страница приветствия'`.
- Остальные страницы (главная, статьи) title не передают → у них «Мой блог».

Переменные попадают в шаблон через `extract($vars)` в методе `View::renderHtml()`.

## Как запустить

**XAMPP / OpenServer / хостинг:** импортировать `database.sql`, проверить `src/settings.php`, указать корнем папку `www`.

**Встроенный сервер PHP:**
```bash
php -S localhost:8000 -t www www/router.php
```

## Адреса для проверки
- `/` — title во вкладке браузера: «Мой блог» (по умолчанию)
- `/articles/1` — title: «Мой блог»
- `/hello/Имя` — title: **«Страница приветствия»** ← задание ДЗ4
